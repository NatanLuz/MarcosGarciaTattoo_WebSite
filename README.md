# Marcos Garcia Tattoo — Portfolio Website

A modern, responsive portfolio website for tattoo artist Marcos Garcia, built with HTML, CSS, and vanilla JavaScript.

### Why this project (for recruiters and hiring managers)
- Demonstrates ability to translate a real client brief into a polished, production-ready UI
- Focus on brand identity: cohesive color system (Black, Charcoal, Suede Brown, Ice White) and visual consistency
- Performance and UX: responsive layout, mobile navigation, smooth interactions, accessible links and clear hierarchy
- Vanilla stack: clean code without heavy frameworks, easy to host and maintain

## ✨ Features
- Responsive design (desktop/tablet/mobile)
- Hero with high-impact image
- About section with studio portrait
- Differentials (value proposition cards)
- Portfolio with category filters (Black & Grey, Color, Minimal/Fineline)
- Reviews slider showcasing client feedback
- Location with professional layout + embedded Google Maps + social/contact shortcuts
- Contact section with Email, Instagram, Facebook and WhatsApp
- Smooth scrolling, soft top gradient, subtle animations

## 📁 Project Structure
```
MarcosGarciaTattoo_WEBSITE/
├── index.html          # Main page
├── styles.css          # CSS styles
├── script.js           # JavaScript interactions
└── README.md           # Documentation
```

## 🚀 Getting Started
1. Open `index.html` in any modern browser
2. Replace images as needed inside the `img/` folder (existing filenames are referenced in the HTML)

## 🖼️ Image Guidelines
- Hero: 1 impactful tattoo (neutral background, strong contrast)
- About: 1 studio photo of Marcos
- Portfolio: 15–20 curated photos (categories prepped for filtering)
- Optional contact photo: humanizes the footer area

## 🗺️ Location & Social
- Google Maps: embedded with the studio address
- Quick links: Facebook, Instagram, WhatsApp, and Email added in both Location and Contact sections

## 🛠️ Tech Notes
- No frameworks, no backend
- Progressive enhancement: works well without JS; JS enhances filters, animations and slider
- Mobile-first responsiveness: careful spacing, grid to single-column transitions

## 🎯 Impact (client and developer value)
- For the client: a cohesive visual identity online, clear showcase of styles, simple booking/contact paths, trust via reviews
- For me: showcases design systems thinking, frontend craftsmanship, accessibility mindset, and ability to deliver real-world briefs end-to-end
- Outcome: improved brand perception, better lead capture via direct WhatsApp/Instagram/email links, and a maintainable codebase
